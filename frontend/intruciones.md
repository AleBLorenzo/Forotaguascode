# Prompt definitivo para Minimax 2.5 – Foro Frontend Responsive (Desktop, Tablet, Mobile)

**Prompt:**

"Design a modern, professional, and fully responsive web forum frontend interface, generating mockups for **desktop, tablet, and mobile simultaneously**. The forum icon is a **stylized palm tree**, and the color palette is inspired by palm trees:

- **Greens:** #2E7D32, #81C784, #A5D6A7  
- **Browns / earth tones:** #6D4C41, #A1887F, #D7CCC8  
- **Accents / highlights:** #FFEB3B, #FFC107  

**Screens to include (all responsive):**

1. **Homepage / Main Feed**
   - Header with forum logo (palm tree), navigation (Home, Categories, Profile, Notifications), search bar, login/signup buttons.
   - Main feed: card-style posts with avatar, username, timestamp, post title, content preview, like/comment icons.
   - Sidebar: trending topics, active users, categories, call-to-action buttons.
   - Footer: About, Contact, Terms, Privacy.

2. **Post View Screen**
   - Full post: title, author, timestamp, content, images.
   - Nested comments: avatars, usernames, timestamps, like buttons, reply option.
   - Sidebar: related posts and categories.

3. **User Profile Screen**
   - Avatar, username, bio, stats (posts, followers, following).
   - User posts in cards with interactive buttons.
   - Settings / edit profile button.

4. **Post Creation / Editor**
   - Text area placeholder: 'Write your post…', upload image option, add tags, submit button.
   - Hover effects and active states visible.

5. **Notifications / Alerts**
   - Dropdown/pop-up style notifications with badges for new comments, likes, mentions.

**Design Guidelines:**
- **Responsive layouts:** Desktop (wide), Tablet (medium), Mobile (narrow), all shown in the mockup.
- **Typography:** Headings: Roboto Bold, Body: Roboto Regular or similar.
- **Modern UI trends:** card-style posts, rounded corners, subtle shadows, clean spacing.
- **Interactive elements visually represented:** buttons, input fields, notifications.
- **Placeholders for content:** User123, Post title, Lorem ipsum…
- **Icons included:** likes, comments, notifications, search, settings.
- Emphasize **visual hierarchy, readability, and intuitive navigation**.
- High-resolution, realistic, professional, clean, tech-inspired style similar to Reddit or Discourse.

**Tags / Keywords:**
`web design, forum UI, responsive, professional, modern, clean, tech, palm tree icon, green palette, brown tones, light theme, card layout, desktop, tablet, mobile, interactive elements, hover states, typography, spacing, visual hierarchy, placeholder content, icons, buttons, input fields, notifications, high resolution, frontend mockup`

**Optional Minimax Parameters:**
`--ar 16:9 --v 5 --style realistic --quality 2 --hd --tile --multi-resolution desktop,tablet,mobile`

---

💡 **Uso recomendado:**
- Genera **tres resoluciones en un solo prompt** para tener referencia de responsive design.
- Los **colores HEX** sirven de guía para paleta de frontend real.
- Los **placeholders y mockups de iconos** permiten implementar el frontend en React/Vite sin datos reales.
- Útil para entregar un diseño completo a desarrolladores o usar como inspiración para CSS y componentes.
